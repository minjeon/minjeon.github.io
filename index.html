// 01 — About Me

자기소개

저는 데이터를 통해 비즈니스의 핵심 질문에 답하는 것을 목표로 하는 데이터 분석가입니다. 수치 너머에 있는 인사이트를 발굴하고, 이를 명확한 논리와 시각적 스토리텔링으로 전달하는 데 집중합니다.

Python과 SQL을 중심으로 데이터 수집, 전처리, 시각화, 모델링까지 전 과정을 수행하며, 통계 분석과 머신러닝 기법을 결합해 실질적인 비즈니스 가치를 창출합니다.

단순히 분석하는 것을 넘어, 데이터 기반 의사결정 문화를 조직에 정착시키는 시니어 데이터 분석가로 성장하고자 합니다.
